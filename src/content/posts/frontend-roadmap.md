## 为什么先补基础

前端基础不是只会写页面，而是理解页面结构、样式规则和浏览器如何渲染内容。

我打算把基础阶段分成三条线：

1. HTML 语义化：标题、段落、列表、表单、可访问性。
2. CSS 布局：Flex、Grid、响应式、主题变量。
3. JavaScript 基础：数据结构、函数、模块、异步。

## 复习方式

每学完一个小知识点，就写一个很短的 demo。比起只看教程，自己动手更容易发现真正没懂的地方。
